# Carine Byande — Site personnel

Site one-page de Carine Byande, Directrice des Opérations IT.

🌐 **En ligne** : [carinebyande.levent-it.com](https://carinebyande.levent-it.com)

## Stack

Site statique en HTML/CSS pur — aucune dépendance, aucun build.

- `index.html` — structure et contenu
- `style.css` — styles (palette ardoise + or)
- `avatar.jpg` — portrait

## Polices

Cormorant Garamond (serif) + Inter (sans-serif) chargées depuis Google Fonts.

## Déploiement

Push sur la branche `main` → Hostinger redéploie automatiquement sur `carinebyande.levent-it.com`.

## Mise à jour du contenu

Pour modifier un texte ou une couleur :

1. Éditer le fichier concerné (`index.html` ou `style.css`)
2. `git commit` + `git push`
3. Sur Hostinger → Avancé → GIT → cliquer "Déployer" pour tirer la dernière version
